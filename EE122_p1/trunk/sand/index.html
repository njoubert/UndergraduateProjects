Bezerker's Home Page

This is the main page server by Bezerker, a web server written completely in C over the course of two consecutive 3 day coding sessions for EE122, a UC Berkeley networking course.
It supports HTTP/1.0 GET requests, and the presistent connections and chunking extentions as implemented before HTTP/1.1 became a standard.
